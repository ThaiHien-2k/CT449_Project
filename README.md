## How to run
  - Open 2 terminal

## First terminal
```sh
cd .\backend\

npm install

npm start
```
Server run in:
www.localhost:3000

## Second terminal
```sh
cd .\frontend\

npm install

npm run dev
```
Client run in:
www.localhost:3001
